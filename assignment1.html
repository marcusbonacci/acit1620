What is HTML?

HTML is a universal web language format meant to standardize the display of information on a website.

You can use HTML to:

  • Display webpage content; such as, text, images, links, videos, etc

  • HTML can be used to view files on a server via links and redirects

  • HTML can be used as a server interface

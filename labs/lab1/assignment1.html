HyperText Markup Language

What is HTML?

HTML is a universal web language meant to standardize the display of information on a website.

You can use HTML to:

  • Display webpage content; such as, text, images, links, videos, etc

  • View files on a server via links and redirects

  • Used as a server interface

  • Call and Email someone
